üìù Question
You are given two cakes. The first cake has N pieces and the second cake has M pieces. Find the maximum number of people who can equally get a whole number of pieces from both cakes, without cutting or wasting any piece.
üí° Approach
To ensure every person gets the same number of pieces of each cake and all pieces are used, we need to divide the cakes equally. This means finding the Greatest Common Divisor (GCD) of N and M. The GCD gives the largest number of people that can divide both cakes equally.
üíª Code (Java)
import java.util.Scanner;

public class GCDCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        int N_pieces = scanner.nextInt();
        int M_pieces = scanner.nextInt();
        int fac = 1;
        int res = 1;

        while (fac <= N_pieces && fac <= M_pieces) {
            if (N_pieces % fac == 0 && M_pieces % fac == 0) {
                res = fac;
            }
            fac++;
        }

        System.out.println(res);
        scanner.close();
    }
}
